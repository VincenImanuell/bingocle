# Bingocle Docs

The documentation site for [Bingocle](https://github.com/VincenImanuell/bingocle) —
an AI-refereed community bingo prediction market on Mantle.

Built with **Next.js 16** (App Router) + **MDX**. Content lives in `content/` as
Markdown; the IA, sidebar, breadcrumbs, and search are driven by `lib/nav.ts`.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000  (redirects to /getting-started/introduction)
```

## Build

```bash
npm run build    # regenerates the search index, then builds 26 static pages
npm start        # serve the production build
```

## How it works

| Piece | Where |
|---|---|
| Page content | `content/<section>/<page>.mdx` |
| Navigation / sidebar / breadcrumbs / prev-next | `lib/nav.ts` (single source of truth) |
| MDX → route mapping | `lib/registry.ts` + `app/[...slug]/page.tsx` |
| Theme (purple starfield, code blocks, TOC) | `app/globals.css` |
| Chrome (top bar, sidebar, TOC, search, code blocks) | `components/docs/` |
| Search index | generated by `scripts/build-search-index.mjs` → `lib/search-index.json` |

### Add a page

1. Create `content/<section>/<slug>.mdx` starting with a single `# Title`.
2. Register it in `lib/registry.ts` (one import thunk) and `lib/nav.ts` (sidebar entry).
3. `npm run dev` — the search index regenerates on the next `build`.

> **MDX note:** these pages avoid raw `<` and `{` in prose (they are JSX-significant).
> Wrap identifiers, types, and comparisons in backticks or fenced code blocks.

## Configuration

- GitHub link: `REPO_URL` in `lib/nav.ts`.
- Fonts: Space Grotesk (display) + Inter (body) + JetBrains Mono (code), via `next/font`.
